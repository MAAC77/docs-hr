# Manual del Administrador

> Versión: 1.0
> Última actualización: 2026-04-30
> Audiencia: Administrador
> Estado: Borrador base

---

## Objetivo

Documentar las funciones de administración y parametrización del sistema.

---

## Alcance

Este manual cubre la administración funcional y operativa de componentes críticos.

---

## Módulos Incluidos

### 1. Dashboard administrador

- visión general
- indicadores de control

### 2. Usuarios

- alta
- actualización
- control de información

### 3. Dispositivos

- registro
- revisión
- control de estado

### 4. Mapeo de usuarios

- relación entre usuario interno y usuario del dispositivo
- validaciones

### 5. Registros de dispositivos

- consulta de registros crudos
- revisión de sincronización

### 6. Configuraciones

- parámetros generales del sistema
- impacto funcional

### 7. Reportes globales

- consulta
- filtros
- exportación

---

## Notas para Completar este Manual

Cada capítulo debe indicar claramente:

1. qué acción puede ejecutar el administrador
2. qué efecto tiene en el sistema
3. qué validaciones debe revisar antes de guardar
4. qué hacer si el resultado no coincide con lo esperado
