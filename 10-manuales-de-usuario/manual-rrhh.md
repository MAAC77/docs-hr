# Manual de Operación para RRHH

---

## Propósito

Este manual está dirigido al personal de Recursos Humanos que opera el sistema en sus tareas diarias.

Su función es ayudarte a:

- revisar asistencia del personal;
- organizar departamentos y asignaciones;
- administrar horarios;
- revisar solicitudes de permiso;
- consultar reportes operativos.

Este documento funciona como portada de navegación. Cada proceso importante tiene su propio manual detallado.

---

## Antes de comenzar

Para usar este manual correctamente, debes:

1. ingresar con una cuenta que tenga el rol `RRHH`;
2. verificar que puedes ver los menús `Organización`, `Gestión de Asistencia` y `Permisos`;
3. confirmar que el sistema te permite abrir las pantallas operativas.

Si alguna opción no aparece, solicita la revisión de permisos antes de continuar.

---

## Ruta general del rol

Cuando ingresas con perfil de RRHH, el sistema te dirige a `Reportes de Asistencia`.

Desde el menú lateral podrás acceder a estas opciones:

1. `Reportes de Asistencia`
2. `Tabla Mensual de Personal`
3. `Departamentos`
4. `Asignaciones de Personal`
5. `Feriados`
6. `Horarios`
7. `Asignaciones de Horario`
8. `Todas las Solicitudes`
9. `Pendientes de Aprobación`
10. `Tipos de Permiso`

## Cuándo usar cada manual

- Usa `Reportes de Asistencia` para revisar registros por día, persona o estado.
- Usa `Tabla Mensual de Personal` para revisar un mes completo de una persona.
- Usa `Departamentos` cuando debas crear o actualizar la estructura organizacional.
- Usa `Asignaciones de Personal` cuando debas mover personas entre departamentos.
- Usa `Feriados` cuando debas registrar fechas que afectan la asistencia.
- Usa `Horarios` cuando debas crear o mantener plantillas de horario.
- Usa `Asignaciones de Horario` cuando debas aplicar horarios a una o varias personas.
- Usa `Solicitudes de Permiso` para revisar el historial general de solicitudes.
- Usa `Pendientes de Aprobación` para atender solicitudes que esperan revisión.
- Usa `Tipos de Permiso` cuando debas configurar los tipos disponibles para el personal.

---

## Recomendaciones importantes

- Antes de registrar datos, revisa el período, la persona y el departamento correctos.
- Si un cambio afecta asistencia u horarios, confirma el resultado en la pantalla correspondiente.
- Si una solicitud cambia de estado, verifica que el historial refleje la acción realizada.
- Si una pantalla no devuelve resultados, revisa primero los filtros antes de asumir que faltan datos.
